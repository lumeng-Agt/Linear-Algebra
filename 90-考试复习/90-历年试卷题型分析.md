# 历年试卷题型分析

本页按当前考试范围重排。旧卷里出现的 1.7 线性空间、第三章函数与线性变换、第五章二次型及其应用不再列入必刷内容。

## 整体结构

历年 A 卷通常为闭卷 120 分钟，结构基本稳定：

| 模块 | 常见分值 | 内容 |
|---|---:|---|
| 选择题 | 15 分 | 概念、性质、反例、快速判定 |
| 填空题 | 15-20 分 | 行列式、秩、距离夹角、特征值 |
| 计算题 | 23-24 分 | 矩阵运算、行列式、逆矩阵或伴随矩阵 |
| 解答题 | 36-46 分 | 极大无关组、线性方程组、特征值与对角化 |
| 证明题 | 6-7 分 | 线性相关、线性无关、基础解系、线性表示 |

2022-2023 年加入判断题，并区分普通班与国际班题目，但核心知识点未变。

## 高频大题

### 1. 极大线性无关组

几乎每年出现。

标准要求：

- 求向量组的秩。
- 给出一个极大无关组。
- 把其余向量用该极大无关组表示。

复习对应：[[02-向量组的秩]], [[08-矩阵的秩与向量组的秩]]

### 2. 线性方程组

常见问法：

- 求通解。
- 含参数时讨论唯一解、无解、无穷多解。
- 已知有解或无穷多解，反求参数。
- 用一个特解加基础解系表示通解。

复习对应：[[05-初等变换与线性方程组]]

### 3. 行列式计算

每年计算题中基本都有。

常用方法：

- 化三角。
- 按零多的行或列展开。
- 提取公因子。
- 使用行列式性质。
- 特殊 n 阶行列式。

复习对应：[[06-行列式与克拉默法则]]

### 4. 逆矩阵、伴随矩阵、矩阵方程

常见问法：

- 求 $A^{-1}$。
- 已知 $|A|$ 求 $|A^*|$、$|A^{-1}|$。
- 解 $AXB=C$、$AX+B=C$ 等矩阵方程。
- 判断矩阵运算等式是否成立。

复习对应：[[07-逆矩阵、伴随矩阵与矩阵方程]]

### 5. 特征值、特征向量、相似、对角化

高频点：

- 求特征值和特征向量。
- 已知特征值求迹、行列式、秩。
- 判断可对角化。
- 求可逆矩阵 $P$ 使 $P^{-1}AP$ 为对角阵。
- 已知相似关系反求参数。

复习对应：[[09-特征值、特征向量与数字特征]], [[10-相似与对角化]]

## 选择填空高频陷阱

| 陷阱 | 正确理解 |
|---|---|
| $AB=0$ 推出 $A=0$ 或 $B=0$ | 错，矩阵乘法有零因子 |
| $AB=AC$ 可消去 $A$ | 只有 $A$ 可逆时才可左乘 $A^{-1}$ |
| $(AB)^T=A^TB^T$ | 错，正确为 $(AB)^T=B^TA^T$ |
| $(AB)^{-1}=A^{-1}B^{-1}$ | 错，正确为 $(AB)^{-1}=B^{-1}A^{-1}$ |
| 两个对称矩阵乘积仍对称 | 不一定，除非可交换 |
| 特征值相同必相似 | 不一定 |
| 任意两个向量线性无关则整个向量组线性无关 | 不一定 |
| 非齐次方程组两个解的任意线性组合还是解 | 不一定，系数和为 1 时才仍是非齐次解 |

## 证明题高频模板

证明题反复围绕：

- 已知 $\alpha_i$ 线性无关，证明某组 $\beta_i$ 线性无关或相关。
- 已知基础解系，证明变换后的向量仍是基础解系。
- 已知非齐次方程组一个特解和齐次基础解系，证明某组向量线性无关。
- 已知“可由某组表示但不能由较小组表示”，推出另一个表示关系。

详见：[[92-证明题模板]]

## 临考优先级

1. 必会：初等行变换、求秩、求极大无关组、解方程组。
2. 必会：行列式性质与计算、逆矩阵、伴随矩阵。
3. 必会：特征值、特征向量、迹、行列式、对角化。
4. 必会：证明题模板和选择题反例。
5. 跳过：1.7 线性空间、第三章、第五章二次型。
