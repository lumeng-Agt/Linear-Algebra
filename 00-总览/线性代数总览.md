# 线性代数总览

这套笔记已按魏福义主编《线性代数》（中国农业出版社）目录重新校准，并按当前考试范围排除：1.7 线性空间、第三章函数与线性变换、第五章二次型及其应用。范围边界见[[考试范围与教材目录]]。

## 主线地图

1. 用[[01-向量与线性关系|向量的线性关系]]建立“能否表示、是否相关”的语言。
2. 用[[02-向量组的秩|秩和极大无关组]]把向量组压缩成最少信息。
3. 用[[03-内积、正交与施密特正交化|内积与正交]]处理长度、角度、距离、标准正交基。
4. 用[[04-矩阵及其运算|矩阵]]统一表示线性关系、方程组和递推模型。
5. 用[[05-初等变换与线性方程组|初等变换]]解决计算问题：化阶梯形、解方程组、求秩、求逆。
6. 用[[06-行列式与克拉默法则|行列式]]判断方阵是否可逆、线性方程组是否唯一解。
7. 用[[07-逆矩阵、伴随矩阵与矩阵方程|逆矩阵与伴随矩阵]]处理矩阵方程和常见恒等变形。
8. 用[[08-矩阵的秩与向量组的秩|矩阵的秩]]连接矩阵、向量组、线性方程组。
9. 用[[09-特征值、特征向量与数字特征|特征值与特征向量]]研究方阵的数字特征。
10. 用[[10-相似与对角化|相似与对角化]]解决特征向量综合题。

## 复习顺序建议

第一轮按概念链走：向量组线性相关 -> 秩 -> 初等行变换 -> 线性方程组。这里是整门课的根。

第二轮按计算链走：行列式 -> 逆矩阵 -> 矩阵方程 -> 特征值 -> 对角化。

第三轮按题型走：直接看[[90-历年试卷题型分析]]，把每类题的判定条件和标准步骤背熟，再用[[91-公式与判定速查]]做临考前检查。
